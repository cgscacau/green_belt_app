# Green Belt Six Sigma - Sistema de Gestão de Projetos DMAIC

Sistema completo para gestão de projetos Six Sigma Green Belt utilizando metodologia DMAIC.

## Tecnologias Utilizadas

- Python 3.8+
- Streamlit
- Firebase (Auth + Firestore)
- Pandas, NumPy, Plotly

## Instalação

1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Configure as variáveis do Firebase
4. Execute: `streamlit run app.py`

## Configuração Firebase

1. Crie um projeto no Firebase Console
2. Configure Authentication (Email/Password)
3. Configure Firestore Database
4. Baixe o service account key
5. Configure as variáveis de ambiente
